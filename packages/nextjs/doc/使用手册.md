# 小白使用手册

本手册旨在给Web3小白 / Web2使用者介绍链上DApp --  ”醒了吗“的用法。

以Chrome浏览器和MetaMask钱包为例。（手机版可以下载MetaMask App，在app自带的区块浏览器进入网站）

DApp链接：https://wakeup03.vercel.app/

----

## MetaMask钱包注册

1. 进入插件商店，搜索"MetaMask"，安装插件。（认准官方，不要下载到假货）

   ![image-20260218211541576](./使用手册.assets/image-20260218211541576.png)
   或者进入官网安装（认准官网地址：https://metamask.io/）
   ![image-20260218211837844](./使用手册.assets/image-20260218211837844.png)

2. 点击“创建新钱包”

   ![image-20260218212630442](./使用手册.assets/image-20260218212630442.png)

   点击“使用私钥助记词”继续

   ![image-20260218212905016](./使用手册.assets/image-20260218212905016.png)

3. 创建密码（此密码记牢，是钱包的登录密码）

   ![image-20260218213207967](./使用手册.assets/image-20260218213207967.png)

4. 保存助记词。

         这是一组有序12个单词，是钱包的唯一凭证（和登陆密码不同）。

         你可以凭助记词在其他设备登陆你的本钱包。其他人若拿到了，也可以转走你的钱包里的钱。

         建议离线保存，不要保存在微信、Cloud等在线数据库，有效降低账户被盗的概率。

   ![image-20260218213717874](./使用手册.assets/image-20260218213717874.png)

5. 完成

   ![image-20260218214934771](./使用手册.assets/image-20260218214934771.png)

## ”醒了吗“连接MetaMask钱包登录

6. 进入网站，点击登录

   ![image-20260218220201071](./使用手册.assets/image-20260218220201071.png)

7. 选择MetaMask方式登录

   ![image-20260218220518524](./使用手册.assets/image-20260218220518524.png)

8. 登陆成功

**恭喜你！登陆成功，现在可以使用网站了！**
